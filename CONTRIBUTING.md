# Contributing

Thank you for considering contributing to the Filament Menu Manager plugin!

1.  Fork the repository.
2.  Clone your fork locally.
3.  Install dependencies: `composer install`.
4.  Create a new branch for your feature or bug fix: `git checkout -b feature/my-feature`.
5.  Make your changes.
6.  Run tests to ensure nothing is broken: `composer test`.
7.  Commit your changes: `git commit -m "Add my feature"`.
8.  Push to your fork: `git push origin feature/my-feature`.
9.  Submit a Pull Request.

## Coding Style

Please follow the [PSR-12](https://www.php-fig.org/psr/psr-12/) coding standard.

## Tests

This package uses [Pest](https://pestphp.com/) for testing. Please ensure all tests pass before submitting a PR.

```bash
composer test
```
